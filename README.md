# watch-movies-rn

> 使用 react-native 开发的iOS APP

> 下载地址: [App Store(用iOS Safari浏览器打开)](https://itunes.apple.com/cn/app/%E7%94%B5%E5%BD%B1%E6%96%B0%E9%A3%8E%E5%90%91/id1382568619)

> 或者在iOS App Store中搜索 “电影新风向” 

## 功能

- 电影列表展示
- 电影详情展示
- 上拉刷新
- 下拉加载更多数据
- 搜索
- 持久化储存搜索历史
- 适配了iPhone X
- 发布到APP Store
